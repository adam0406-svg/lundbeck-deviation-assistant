Built for Lundbeck application, 2026

Quality teams in pharmaceutical manufacturing handle deviations daily, from documentation gaps to critical contamination events. Each one requires structured investigation, severity classification, root cause analysis, and corrective action planning. The documentation is rigorous, repetitive in structure, and time-consuming.

This prototype demonstrates how generative AI can assist that process. It takes structured deviation input and generates a first-draft investigation report with severity classification, Ishikawa-based root cause investigation areas, draft CAPA recommendations, and regulatory cross-references. The AI handles the structure. The quality expert provides the substance.

Built on the ISPE methodology for integrating generative AI into deviation management
Structure, Not Substance AI generates the framework and documentation structure. Domain expertise, root cause determination, and quality decisions remain with the investigator.
Human-in-the-Loop by Design Every output section is marked for human review. The system classifies but never closes. It recommends but never approves. GxP compliance requires human oversight, and this is built into the architecture.
Context Drives Quality Structured input fields ensure the AI receives the right context for each deviation. Generic prompts produce generic output. Specific, structured questions produce actionable drafts.
Honest About Limits When information is missing, the system flags it explicitly rather than fabricating details. Uncertainty is stated, not hidden. Conservative classification is the default.

1 Deviation Summary

On 2026-04-28 at 14:30, during routine batch record review for Escipra Tablets 10 mg (Batch EX-2026-0412), a missing second-person verification signature was identified on the weighing step for excipient Lactose Monohydrate (line 14 of the batch record). The operator signature was present, and the correct weight was confirmed through cross-referencing with the dispensing log and scale printout. The batch has not progressed beyond the dispensing stage.

2 Severity Classification

Minor

This deviation represents a documentation gap in a procedural requirement (dual-signature verification) rather than an error in the actual weighing operation. The correct weight has been independently confirmed via the dispensing log and scale printout. There is no indication that product quality or patient safety is impacted. However, the missing signature represents a failure to follow SOP-RM-003 Section 4.2, which constitutes a GMP documentation deficiency.

⚠ HUMAN REVIEW REQUIRED: Severity classification must be reviewed and confirmed by the QA reviewer. Consider whether this is an isolated occurrence or part of a pattern of documentation lapses for this operator, shift, or process step.

3 Impact Assessment

Product Quality

Low risk. The actual weighing was performed correctly as confirmed by independent records (dispensing log, scale printout). The excipient quantity in the batch is not in question.

Patient Safety

No indication of patient safety risk. The correct amount of Lactose Monohydrate was dispensed.

Batch Disposition

The batch is currently on QA hold at the dispensing stage. Given the independent confirmation of correct weighing, batch progression may be considered following completion of the deviation investigation and appropriate documentation correction.

Scope

Assess whether this is an isolated event. Review recent batch records from the same operator, shift, and dispensing area for similar documentation gaps. If a pattern is identified, the severity classification should be escalated to Major.

⚠ HUMAN REVIEW REQUIRED: Batch disposition decision must be made by authorized QA personnel. Review whether the existing independent records (dispensing log, scale printout) provide sufficient documented evidence to support batch progression.

● Process / Method

  • Does SOP-RM-003 clearly define when the verification signature must be applied (immediately after verification, or can it be batched)?
  • Is the batch record layout clear regarding which lines require dual signatures?
  • Has this SOP been recently revised, and if so, were all personnel retrained?

● Equipment / Machine

N/A. Not equipment-related.

● Material

N/A. Not material-related.

● Measurement

N/A. The weighing itself was performed correctly.

● Environment

  • Were there any environmental factors (noise, congestion, shift change) in the dispensing area that may have contributed to the oversight?
  • Is the dispensing area layout conducive to completing documentation at the point of activity?

Event
Environmental monitoring alarm: Cold Storage Room CS-02 reached 9.2°C. Validated range 2–8°C. Excursion lasted ~2h 45min. Contains API intermediates for 3 batches and 12 reference standards.
Discovered
2026-04-28, 06:00 (morning shift review of overnight alarms)
Product / Batch
Multiple: Batches LU-2026-0087, LU-2026-0088, LU-2026-0091 (API intermediates) + 12 QC reference standards
Process Step
In-process storage of API intermediates between synthesis and formulation
Expected
2–8°C continuously per validated storage conditions
Observed
9.2°C (1.2°C above limit) for ~2h 45min (03:17–06:02). Gradual rise from ~02:30.
Immediate Actions
All materials quarantined. Facilities dispatched; condenser fan found stopped. Backup cooling activated. Temperature recovered to 5.1°C by 06:45. QC notified regarding reference standards.

1 Deviation Summary

On 2026-04-28, a temperature excursion was detected in Cold Storage Room CS-02, where the recorded temperature reached 9.2°C, exceeding the validated storage range of 2–8°C by 1.2°C. Continuous monitoring data indicates the excursion began at approximately 02:30 with the alarm triggering at 03:17, and the temperature returned to within specification at 06:02, resulting in an excursion duration of approximately 2 hours and 45 minutes. CS-02 contained API intermediates for Batches LU-2026-0087, LU-2026-0088, and LU-2026-0091, as well as 12 reference standards used in QC testing. Preliminary inspection identified a stopped condenser fan as the likely cause.

2 Severity Classification

Major

This deviation represents a significant departure from validated storage conditions for API intermediates and reference standards. The excursion magnitude (1.2°C above the upper limit) and duration (2 hours 45 minutes) are sufficient to warrant investigation into potential impact on material quality, particularly for temperature-sensitive intermediates and reference standards. While the excursion was not extreme, the number of materials affected (three batches of API intermediates plus 12 reference standards) increases the potential scope of impact. Escalation to Critical should be considered if stability data indicates material degradation.

⚠ HUMAN REVIEW REQUIRED: Severity classification must be reviewed and confirmed by the QA reviewer. If stability data for the affected materials indicates sensitivity to the excursion conditions (9.2°C for ~3 hours), escalation to Critical may be warranted.

3 Impact Assessment

Product Quality

The primary concern is whether the temperature excursion affected the chemical stability, physical properties, or potency of the stored API intermediates. The impact depends on the specific stability profiles of each material at the excursion temperature. Reference standard integrity must also be assessed, as compromised reference standards would affect the validity of all QC test results generated using them.

Patient Safety

Indirect risk. If the API intermediates were degraded by the excursion and this went undetected, it could result in subpotent or impure finished product. If reference standards were compromised and continued to be used, QC test results would be unreliable. Both scenarios present potential patient safety risk if not properly investigated.

Batch Disposition

Three batches of API intermediates (LU-2026-0087, LU-2026-0088, LU-2026-0091) should remain quarantined until the impact assessment is complete. Twelve reference standards must be assessed for continued suitability before use in further testing.

Scope

Assess whether CS-02 has had previous temperature excursions (trend analysis). Determine whether other cold storage rooms share the same cooling system or maintenance schedule. Evaluate whether the condenser fan failure indicates a broader equipment maintenance gap.

⚠ HUMAN REVIEW REQUIRED: Impact assessment requires review by qualified personnel with knowledge of the specific stability profiles of the affected materials. Batch disposition decisions must be made by authorized QA personnel.

4 Root Cause Investigation Plan

● Personnel

  • Who was responsible for monitoring overnight alarms? Was there a delay in responding to the alarm at 03:17?
  • What is the escalation procedure for after-hours environmental alarms, and was it followed?
  • Was the morning shift operator who discovered the deviation at 06:00 the first person to check, or was the alarm acknowledged earlier without action?

● Process / Method

  • What is the alarm response SOP for environmental excursions? Does it define maximum response time?
  • Is there an automated escalation process (SMS/call to on-call personnel) for critical alarms, or does it rely on manual monitoring?
  • When was the last preventive maintenance performed on the CS-02 cooling system? Is the PM schedule adequate?

● Equipment / Machine

  • What caused the condenser fan to stop? Electrical failure, mechanical wear, control system issue?
  • When was the condenser fan last inspected, serviced, or replaced?
  • Is there redundancy in the cooling system for CS-02? If so, why did the backup not activate automatically?
  • Review calibration records for the temperature monitoring sensors in CS-02.

● Material

  • What are the validated storage conditions and stability data for each affected API intermediate at temperatures above 8°C?
  • Do the material specifications include data for short-term excursions at 9–10°C?
  • What is the remaining shelf life of each reference standard? Are any near expiry and more susceptible to degradation?

● Measurement

  • Confirm accuracy of the temperature monitoring system. Cross-reference the reading with an independent calibrated probe if available.
  • Review continuous monitoring data for the period leading up to the excursion. Was it a gradual warming trend or a sudden event?

● Environment

  • Was there any external factor affecting CS-02 temperature (adjacent equipment, HVAC issues, door left open)?
  • Review door access logs for CS-02 during 02:00–06:00.
  • Were there facility-level power fluctuations or outages that could have affected the cooling unit?
⚠ HUMAN REVIEW REQUIRED: The investigator determines which areas are relevant and conducts the actual investigation. Equipment root cause analysis should involve Facilities/Engineering.

5 Draft CAPA Recommendations

Corrective Actions
1. Complete equipment repair and verification. Repair or replace the failed condenser fan. Confirm sustained performance within 2–8°C before returning CS-02 to routine use. (Owner: Facilities/Engineering; may require requalification)
2. Conduct stability impact assessment. Review stability data for each affected API intermediate to determine whether the excursion conditions fall within acceptable limits. Consider retesting if data is insufficient. (Owner: QC / Stability)
3. Assess reference standard integrity. Retest or replace the 12 reference standards stored in CS-02 during the excursion. (Owner: QC)
4. Determine batch disposition. Based on stability assessment results, make disposition decisions for affected batches. (Owner: QA)
Preventive Actions
1. Review and upgrade alarm escalation procedures. Evaluate whether after-hours alarm response is adequate. Consider automated SMS/call escalation with defined maximum response times. (Owner: QA / Facilities; requires change control)
2. Review preventive maintenance schedule. Assess whether PM frequency and scope are adequate to prevent condenser fan failures. (Owner: Facilities/Engineering; requires change control)
3. Evaluate cooling system redundancy. Assess whether CS-02 should have automatic backup cooling to prevent excursions from single-point equipment failures. (Owner: Facilities/Engineering; capital investment may be required)
4. Trend analysis of environmental excursions. Review 12 months of temperature monitoring data across all cold storage rooms. (Owner: QA)
⚠ HUMAN REVIEW REQUIRED: CAPA recommendations are suggestions only. Final plan must be developed with input from Facilities, QC, and QA. Equipment modifications and SOP revisions require formal change control.

Cleaning verification failure post-changeover on Tablet Press TP-03. Swab result: 8.4 µg/cm² Nortriptyline HCl residue on punch faces (limit: ≤ 4.0 µg/cm²). Changeover from Nortriptyline 25 mg to Citalopram 20 mg. Cleaning signed off as complete. Product B not yet manufactured.
Discovered
2026-04-28, 07:15
Product / Batch
Product B: Citalopram 20 mg, Batch CT-2026-0203 (not started). Previous: Nortriptyline HCl 25 mg, Batch NT-2026-0195 (quarantine pending release).
Process Step
Equipment cleaning verification: post-changeover, pre-production check
Expected
≤ 4.0 µg/cm² residue per Cleaning Validation Protocol CVP-2024-018
Observed
Punch faces: 8.4 µg/cm² (210% of limit). Hopper: 1.2 µg/cm² (pass). Feed frame: 0.8 µg/cm² (pass).
Immediate Actions
TP-03 on hold. Re-cleaning initiated with additional manual scrubbing. QA hold on previous batch NT-2026-0195. Cleaning team lead notified. All concurrent changeover equipment flagged for review.

1 Deviation Summary

On 2026-04-28 at 07:15, cleaning verification following a product changeover on Tablet Press TP-03 revealed that swab test results from punch face surfaces returned a residue level of 8.4 µg/cm² for the previous product (Nortriptyline HCl 25 mg), exceeding the validated cleaning limit of 4.0 µg/cm² by 110%. The changeover was from Nortriptyline HCl 25 mg tablets (Batch NT-2026-0195) to Citalopram 20 mg tablets (Batch CT-2026-0203). Cleaning was performed per SOP-CL-011 and signed off as complete. The deviation was caught during pre-production verification, and Product B manufacturing had not commenced. Swab samples from the hopper (1.2 µg/cm²) and feed frame (0.8 µg/cm²) were within specification.

2 Severity Classification

Critical

This deviation involves a cleaning verification failure for a product changeover between two different active pharmaceutical ingredients (Nortriptyline HCl and Citalopram). Residual Nortriptyline on product contact surfaces at 210% of the validated limit represents a confirmed cross-contamination risk. Although the deviation was detected before Product B manufacturing commenced, the fact that cleaning was performed, signed off as complete, and yet failed verification indicates a breakdown in the cleaning process that could have resulted in cross-contamination if the pre-production verification step had been missed. The previous batch (NT-2026-0195) warrants retrospective review.

⚠ HUMAN REVIEW REQUIRED: Severity classification must be reviewed and confirmed by the QA reviewer. Critical classification is recommended due to cross-contamination risk between two active pharmaceutical ingredients, even though actual contamination of Product B was prevented.

3 Impact Assessment

Product Quality

Product B (Citalopram, Batch CT-2026-0203): Not affected. Manufacturing had not commenced. No product quality impact, provided re-cleaning is verified as effective before production starts.

Product A (Nortriptyline, Batch NT-2026-0195): Not directly affected by this cleaning failure. However, the failure raises questions about whether previous changeovers from other products TO Nortriptyline were adequately cleaned. Retrospective review may be warranted.

Cleaning process integrity: The primary quality concern is that the cleaning process (SOP-CL-011) was signed off as complete but failed analytical verification. This indicates the cleaning procedure, execution, or verification process has a gap.

Patient Safety

The cross-contamination risk between Nortriptyline (a tricyclic antidepressant) and Citalopram (an SSRI) is clinically significant. Co-exposure to both compounds could cause serious adverse effects including serotonin syndrome. In this case, patient safety was protected by the pre-production verification step.

Batch Disposition

Batch CT-2026-0203 (Citalopram): Not yet manufactured. Hold production until TP-03 passes re-cleaning verification. Batch NT-2026-0195 (Nortriptyline): Maintain quarantine until investigation confirms no retrospective concerns.

Scope

Assess whether this was isolated or could have occurred on previous changeovers on TP-03 or other tablet presses. Review cleaning verification results for the last 12 months on TP-03, particularly changeovers involving Nortriptyline HCl. Assess other equipment cleaned using SOP-CL-011 in the same period.

⚠ HUMAN REVIEW REQUIRED: Impact assessment requires review by qualified personnel with knowledge of the specific products, cleaning validation protocols, and toxicological limits. The clinical significance of cross-contamination risk should be confirmed by the site toxicologist or pharmacovigilance team.

4 Root Cause Investigation Plan

● Personnel

  • Who performed the cleaning? Review training records for SOP-CL-011 and cleaning verification procedures.
  • Has this operator performed changeover cleaning before? What were prior results?
  • Was cleaning performed during a shift change, under time pressure, or with interruptions?
  • Who signed off cleaning as complete? Was sign-off based on visual inspection only, or procedural verification?

● Process / Method

  • Does SOP-CL-011 include specific instructions for punch face cleaning (disassembly, manual scrubbing, soaking)?
  • Is the procedure validated specifically for Nortriptyline HCl residue removal? When was cleaning validation last performed?
  • Is there a difference in cleaning difficulty between punch faces and other components (hopper and feed frame passed)?
  • Does the procedure require disassembly of punches, or are they cleaned in place?

● Equipment / Machine

  • What is the condition of the punch face surfaces? Scratches, pitting, or wear that could trap residue?
  • When were the punches last replaced or refurbished?
  • Are punch faces more difficult to clean due to geometry (engravings, embossing, break lines)?
  • Was cleaning equipment (brushes, spray nozzles) worn or malfunctioning?

● Material

  • Is Nortriptyline HCl known to be difficult to remove from stainless steel surfaces?
  • Is the cleaning agent effective for Nortriptyline HCl removal? Solubility data available?
  • Were there changes to the Nortriptyline formulation that could affect cleaning difficulty?

● Measurement

  • Review swab sampling technique. Was the sampling location on the punch faces representative?
  • Confirm analytical method is validated for Nortriptyline HCl at the relevant concentration range.
  • Were swab recovery factors accounted for in the 8.4 µg/cm² result?

● Environment

  • Were environmental conditions (cleaning water temperature, humidity, drying time) adequate?
  • Was cleaning performed in the designated wash area or on the production floor?
⚠ HUMAN REVIEW REQUIRED: The investigator determines which areas are relevant. Root cause may involve a combination of factors. Engineering and cleaning validation specialists should be involved.

5 Draft CAPA Recommendations

Corrective Actions
1. Complete re-cleaning and verification of TP-03. Re-clean all product contact surfaces with additional manual scrubbing of punch faces. Do not release for Product B until swab results confirm ≤ 4.0 µg/cm². (Owner: Production / QA)
2. Retrospective review of recent changeovers on TP-03. Review all changeover cleaning verification results for the last 6–12 months, focusing on changeovers involving Nortriptyline HCl as the preceding product. (Owner: QA)
3. Assess other equipment cleaned in the same period. Review whether other tablet presses cleaned during the same timeframe (or by the same operator) should be re-verified. (Owner: QA / Production)
4. Hold decision on Batch NT-2026-0195. Maintain QA hold until retrospective review confirms no concerns about prior cleaning adequacy. (Owner: QA)
Preventive Actions
1. Review and revise cleaning procedure for punch faces. Evaluate whether SOP-CL-011 requires enhanced steps for punch face surfaces, including disassembly, soaking, or additional scrubbing cycles. Update and revalidate if warranted. (Owner: Cleaning Validation / QA; requires change control and revalidation)
2. Evaluate punch surface condition program. Implement or review a punch inspection and replacement program. Worn surfaces may trap residue. (Owner: Engineering / Production; may require change control)
3. Assess cleaning agent effectiveness. Evaluate whether an alternative cleaning agent or enhanced parameters would improve Nortriptyline HCl residue removal. (Owner: Cleaning Validation; requires change control and revalidation)
4. Reinforce cleaning verification as a critical step. Ensure personnel understand that sign-off confirms procedural completion only. Analytical verification (swab testing) is the definitive measure of cleaning adequacy. Consider adding a checklist prompt. (Owner: QA / Training; requires change control if SOP modified)
⚠ HUMAN REVIEW REQUIRED: CAPA recommendations are suggestions only. Final plan must be developed with input from Cleaning Validation, Engineering, QC, and QA. SOP revisions and cleaning procedure changes require formal change control and may trigger revalidation.

What This Prototype Does Not Do

Not a Decision-Maker

This system does not close deviations, approve batch disposition, determine root causes, or make regulatory submission decisions. Every output is a draft requiring qualified human review. In a GxP environment, AI assists. It does not decide.

Not Context-Aware (Yet)

This prototype has no access to a Quality Management System, historical deviation data, or site-specific SOPs. A production version would use retrieval-augmented generation (RAG) to connect to these systems, enabling trend analysis and site-specific recommendations. The architecture is designed to support this.
